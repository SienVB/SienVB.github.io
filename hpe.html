Hybrid IT management framework

From the first of March 2021 until the 28th of March 2021 I did an internship at Hewlett Packard Enterprise in Diegem, Belgium. During this internship I was part of the Hybrid IT team, and developed a project in this subject. This project consisted of building a solution to provide centralized server- and identity management, and authentication in a hybrid IT environment. Which means creating and documenting a proper and efficient solution to manage identities (users/people) and systems (servers and clients) in an environment that consists of both systems on-premises (physical), and systems that are in the cloud of Microsoft Azure (virtual). The solution also provides some other functionalities like user authorization, failover support, role based access control, etc.

This solution would serve as a framework that can be used fully/partly to create solutions for organizations that want to migrate or expand their on-premise systems to a cloud environment. It is built with best practices in mind, and while making use of Microsoft's Cloud Adoption Framework.
